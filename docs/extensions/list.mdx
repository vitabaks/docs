# Extension List

To extend the functionality of the PostgreSQL database, you can use extensions.

:::info
**Over 400 extensions** are available for [installation](/docs/extensions/install), including contrib extensions bundled with PostgreSQL, as well as third-party extensions from the official PGDG repository and others, such as the [Pigsty extension repository](https://ext.pigsty.io).
:::
